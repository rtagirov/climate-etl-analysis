ETL и анализ климатических данных 
(NetCDF — SQLite — Time Series)

Описание проекта

В проекте реализован полный пайплайн обработки климатических данных в формате NetCDF:
загрузка многомерных данных (xarray)
преобразование в табличный формат
сохранение в SQLite базу
SQL-агрегации
анализ временных рядов
визуализация трендов и сезонности

Проект демонстрирует навыки Data Analyst: 
ETL, SQL, Python, работа с временными рядами.

Используемые технологии

Python
pandas
xarray
NetCDF4
SQLite
matplotlib

Этапы пайплайна

1. Загрузка NetCDF
чтение многомерного датасета
анализ координат и переменных

2. Преобразование
выбор переменной температуры
разворачивание в табличный формат
формирование DataFrame

3. Хранение
запись в SQLite
проверка целостности данных

4. SQL-анализ
агрегация по широте
агрегация по времени
группировки

5. Feature Engineering
извлечение month / day / weekday
rolling mean (скользящее среднее)

6. Визуализация
температура по широте
временной тренд
сглаживание тренда

Как запустить

pip install -r requirements.txt
jupyter-notebook da_portfolio_pipeline.py
