# Multidimensional Climate Data Processing Pipeline

Демонстрация полного цикла обработки многомерных климатических данных в формате NetCDF с использованием Python.

Показ практических навыков:
- работы с научными форматами данных,
- преобразования многомерных массивов,
- построения аналитического pipeline,
- хранения и анализа данных через SQL,
- визуализации и интерпретации результатов.

---

## Используемый датасет

- файл: air_temperature.nc
- формат: NetCDF
- тип: учебный климатический датасет
- источник: NCEP / NOAA Reanalysis
- переменная: температура воздуха
- временное разрешение: 6 часов
- координаты и структура: time × latitude × longitude

Выбор датасета обусловлен его компактным размером и удобством для локальной обработки.

---

## Цель проекта

Показать, как можно:
- работать с NetCDF-данными в Python,
- извлекать и агрегировать многомерные данные,
- преобразовывать их в табличный вид,
- хранить данные в SQL-базе,
- выполнять аналитические запросы,
- строить интерпретируемые визуализации.

---

## Стек технологий

- Python
- NumPy
- Pandas
- Xarray
- Matplotlib
- SQLite

---

## Этапы

1. Загрузка и исследование NetCDF-датасета
2. Работа с координатами и измерениями
3. Формирование подвыборки данных
4. Преобразование в pandas DataFrame
5. Сохранение данных в SQLite
6. Аналитические SQL-запросы
7. Визуализация:
   - температура по широте
   - температура во времени
   - агрегации по дням недели
   - сглаживание временного ряда (rolling mean)#
